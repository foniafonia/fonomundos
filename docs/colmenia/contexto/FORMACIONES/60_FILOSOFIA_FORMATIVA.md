# Filosofía formativa

Las formaciones de José no buscan enseñar herramientas.
Buscan cambiar la manera de pensar del profesional.

La IA cambia constantemente.
El criterio permanece.

Objetivos:
- Ahorrar tiempo.
- Mejorar el razonamiento clínico.
- Integrar la IA en la práctica diaria.
- Evitar el uso superficial de prompts.
