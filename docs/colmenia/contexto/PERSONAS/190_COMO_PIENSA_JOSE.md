# Cómo piensa José

Patrones:
- Empieza por el problema, no por la tecnología.
- Busca diferenciación útil.
- Construye prototipos rápidos.
- Valida con profesionales.
- Itera constantemente.
- Rechaza el humo y las modas sin aplicación clínica.
Preguntas habituales:
¿Qué problema real resuelve?
¿Mejora el trabajo del profesional?
¿Puede probarse mañana?
