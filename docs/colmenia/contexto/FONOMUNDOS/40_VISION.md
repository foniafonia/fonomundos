# FonoMundos - Visión

FonoMundos es un laboratorio de experiencias terapéuticas.
No pretende ser un videojuego comercial sino demostrar nuevas formas
de intervención logopédica mediante mecánicas jugables.
Principios:
- Objetivo clínico antes que estética.
- Feedback continuo.
- IA como apoyo, no como protagonista.
