# Herramientas de IA

## Filosofía de uso

Cada IA debe usarse para lo que mejor hace. No hay que casarse con una herramienta.

## Roles sugeridos

### ChatGPT

Estrategia, continuidad, generación de contenido, razonamiento, estructuración de proyectos, prompts, documentos y acompañamiento general.

### Claude

Lectura de grandes contextos, redacción larga, análisis profundo, documentos estratégicos, revisión de estructura y síntesis extensa.

### Codex / Claude Code

Implementación técnica, repositorios, desarrollo, debugging, arquitectura, prototipos y automatizaciones.

### Gemini / NotebookLM

Exploración documental, análisis de fuentes, trabajo con materiales largos, estudio y organización de conocimiento.

### Fable / herramientas visuales

Diseño conceptual, identidad visual, exploración de estética, versiones de interfaz y prototipos visuales.

## Regla de coordinación

José necesita que las IAs trabajen como equipo, no como voces aisladas. Cada una debe recibir contexto suficiente y tareas delimitadas.

## Riesgo

Morir de tokens, repetir contexto, perder decisiones y que cada IA vuelva a proponer lo ya descartado.

Este repositorio existe para evitar eso.
