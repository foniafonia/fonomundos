# Roadmap 2026-2028

## 2026: ordenar y validar

Objetivos:

- Consolidar Telegram como núcleo.
- Lanzar o pre-lanzar COLMENIA.
- Ordenar formaciones iniciales.
- Definir arquitectura de plataforma.
- Seguir mejorando FonoMundos.
- Crear productos digitales pequeños.
- Validar mentorías.
- Sustituir ClinicCloud con MVP funcional si es viable.

## 2027: profesionalizar y escalar

Objetivos:

- Convertir COLMENIA en membresía estable.
- Tener itinerarios formativos claros.
- Desarrollar herramientas propias.
- Mejorar sistema de captación.
- Crear calendario editorial sostenible.
- Estabilizar ingresos recurrentes.
- Diferenciar líneas para logopedas y maestros.

## 2028: consolidar ecosistema

Objetivos:

- Comunidad profesional madura.
- Herramientas propias consolidadas.
- Formación reconocible.
- Posible certificación o método propio.
- Alianzas institucionales.
- Mayor automatización clínica y educativa.

## Prioridad inmediata

No construir más piezas sin ordenar las existentes. El siguiente gran paso debe ser documentación, arquitectura de oferta y lanzamiento controlado.
